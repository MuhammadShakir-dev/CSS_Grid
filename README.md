# What is CSS Grid?
CSS Grid is a new 2 dimensional layout which will help you in handling both Rows and Columns at onces.

# Benifits.
1. CSS Grid will totally replace CSS Float property.
2. It will fasten your coding speed.
3. It will inscrese the readibility of your code.
4. Less coding required.
5. No Need of using CSS Framworkd.
